📄 5. js/video-editor.js – ฟังก์ชันหลัก Video Editor
============================================================
หน้าที่:
- จัดการการเล่นวิดีโอ (Play/Pause/Seek)
- ควบคุม Canvas และ Filter
- ตั้งค่า In/Out Point
- ตัดเฉพาะช่วง (Trim) พร้อมเสียง
- ส่งออกวิดีโอ (Export)
- แสดงผลไทม์ไลน์ (Timeline)

ตัวแปรสำคัญ (State):
- isPlaying        : สถานะเล่น/หยุด
- currentFilter    : ค่า Filter ปัจจุบัน
- inPoint          : จุดเริ่มต้น (วินาที)
- outPoint         : จุดสิ้นสุด (วินาที)
- clips            : รายการคลิปบนไทม์ไลน์

ฟังก์ชันหลัก (API):
- loadVideo(e)                : โหลดวิดีโอจากเครื่อง
- togglePlay()                : เล่น/หยุดวิดีโอ
- resetVideo()                : รีเซ็ตวิดีโอไปที่ 0
- setInPoint()                : ตั้ง In ที่ตำแหน่งปัจจุบัน
- setOutPoint()               : ตั้ง Out ที่ตำแหน่งปัจจุบัน
- applyInOutFromInput()       : ตั้ง In/Out จากช่องตัวเลข
- applyEffect(type)           : ใส่ฟิลเตอร์ (cinematic, fast ฯลฯ)
- trimToSelection()           : ตัดเฉพาะช่วง In-Out (พร้อมเสียง)
- exportTrimmed()             : บันทึกเฉพาะช่วง In-Out เป็นไฟล์
- renderClips(clipData)       : แสดงคลิปบน Timeline
- getVideo(), getCanvas()     : ให้ AI Module เข้าถึง
- setStatus(msg)              : เปลี่ยนข้อความสถานะ

ฟังก์ชันภายใน:
- resizeCanvas()      : ปรับขนาด Canvas ให้พอดี
- startRenderLoop()   : วนลูปแสดงผล (60fps)
- buildFilter()       : สร้าง CSS Filter string
- updateTimeDisplay() : อัปเดตเวลา
- formatTime(t)       : แปลงวินาทีเป็น mm:ss
- updatePlayhead()    : อัปเดตตำแหน่ง Playhead

📄 6. js/ai-module.js – AI สำหรับ Video
============================================================
หน้าที่:
- ตรวจจับฉากด้วย Histogram (เร็ว)
- ตรวจจับฉากด้วย COCO-SSD (แม่นยำ)
- ประมวลผลคำสั่ง Prompt
- สร้างคำบรรยายจำลอง (Captions)

ตัวแปรสำคัญ:
- model             : โมเดล COCO-SSD
- modelLoaded       : สถานะโหลดโมเดล
- _cachedScenes     : Cache ผลลัพธ์ (ใช้ซ้ำ)

ฟังก์ชันหลัก:
- loadAIModel()                       : โหลดโมเดล COCO-SSD
- autoDetectScenesHistogram()         : ตรวจจับฉากด้วย Histogram
- autoDetectScenesML()                : ตรวจจับฉากด้วย ML (COCO-SSD)
- runAI()                             : ประมวลผลคำสั่ง Prompt
- extractNumbersFromPrompt()          : ดึงตัวเลขจากข้อความ (ใช้กับ trim)
- generateCaptions()                  : สร้างคำบรรยายจำลอง

Prompt Engine (runAI):
- ตรวจจับคำสั่ง 'trim' + ตัวเลข → ตั้ง In/Out และตัด
- ตรวจจับ 'cut' / 'ตัด' → เรียก Scene Detection
- ตรวจจับชื่อฟิลเตอร์ → เรียก applyEffect()
- ตรวจจับ 'captions' → เรียก generateCaptions()
- ตรวจจับ 'in' และ 'out' → ตั้ง In/Out

📄 7. js/image-editor.js – ฟังก์ชัน Image Editor
============================================================
หน้าที่:
- แสดงและเลือกภาพตัวอย่าง
- อัปโหลดรูปจากเครื่อง
- ใส่ฟิลเตอร์ตามคำสั่ง (Prompt)
- ดาวน์โหลดรูปที่แก้ไขแล้ว

ตัวแปรสำคัญ:
- ASSETS           : รายการรูปตัวอย่าง 6 รูป
- currentFilter    : ฟิลเตอร์ปัจจุบัน
- currentTab       : แท็บที่เลือก (magic/filter/adjust)
- isProcessing     : กำลังประมวลผล
- filterHistory    : ประวัติฟิลเตอร์ (Undo)

ฟังก์ชันหลัก:
- initImageEditor()    : เริ่มต้น Image Editor
- initGrid()           : สร้าง Grid รูปตัวอย่าง
- setTab(el, type)     : เปลี่ยนโหมด (Magic/Filter/Adjust)
- resetImage()         : รีเซ็ตรูปและฟิลเตอร์
- applyMagic()         : ประมวลผลคำสั่ง Prompt
- processPrompt(text)  : แปลงข้อความเป็นฟิลเตอร์
- applyFilter(filterName) : ใส่ฟิลเตอร์ให้รูป
- downloadImage()      : ดาวน์โหลดรูปเป็น PNG
- handleUpload(file)   : อัปโหลดรูปจากเครื่อง

ฟิลเตอร์ที่รองรับ (12 แบบ):
cinematic, vintage, cyberpunk, bw, pastel, dark, bright, warm, cool, contrast, blur, dramatic

📄 8. js/remove-bg.js – ฟังก์ชันลบพื้นหลัง
============================================================
หน้าที่:
- อัปโหลดรูปหลักและพื้นหลัง
- เปลี่ยนสีพื้นหลัง
- ปรับขนาดวัตถุ (Scale)
- ลากวัตถุ (Drag)
- ลบพื้นหลังด้วย AI (ใช้ @imgly/background-removal)
- ดาวน์โหลดผลลัพธ์

ตัวแปรสำคัญ:
- W, H           : ขนาด Canvas (1200x1200)
- mainImage      : รูปหลัก
- bgImage        : รูปพื้นหลัง
- mainX, mainY   : ตำแหน่งวัตถุ
- mainScale      : ขนาดวัตถุ (0-1)
- mainHasAlpha   : รูปมีอัลฟา (โปร่งใส) หรือไม่
- isDragging     : กำลังลาก

ฟังก์ชันหลัก:
- render()                           : วาดภาพลง Canvas
- getMainDrawParams()                : คำนวณตำแหน่ง/ขนาดวัตถุ
- checkAlpha(image)                  : ตรวจสอบว่ารูปมีอัลฟาหรือไม่
- updateScale()                      : ปรับขนาดวัตถุจาก Slider
- resetPositionAndScale()            : รีเซ็ตตำแหน่ง/ขนาด
- removeBackgroundFromMain()         : ลบพื้นหลังด้วย AI
- refineEdges(imageData, radius)     : ปรับขอบให้เนียนขึ้น
- downloadImage()                    : ดาวน์โหลดเป็น PNG
- onDragStart/Move/End()             : จัดการการลากวัตถุ

การทำงานของ AI Remove Background:
1. แปลงรูปเป็น Canvas
2. ลดขนาดลง (max 512px) เพื่อความเร็ว
3. เรียกใช้ @imgly/background-removal (โมเดล 'small')
4. แปลงผลลัพธ์เป็น Image
5. ปรับขอบให้เนียนขึ้นด้วย Gaussian Blur (refineEdges)
6. แทนที่รูปหลัก และตั้งค่า mainHasAlpha = true
7. Render ใหม่

🔄 การทำงานร่วมกันของระบบ
============================================================

1. ผู้ใช้เปิด index.html → แท็บ Video Editor แสดงเป็นค่าเริ่มต้น
2. คลิกแท็บ → เริ่มต้น Editor ที่เกี่ยวข้อง (ถ้ายังไม่เริ่ม)
3. แต่ละ Editor มีฟังก์ชันเฉพาะ:
   - Video Editor: ใช้ video-editor.js + ai-module.js
   - Image Editor: ใช้ image-editor.js
   - Remove BG: ใช้ remove-bg.js + script1.js
4. AI Module (ai-module.js) ถูกเรียกจาก Video Editor เมื่อต้องการตรวจจับฉาก
5. ทุกฟังก์ชันสามารถแสดง Toast Notification ได้ (ใช้ร่วมกัน)
6. ระบบรองรับ Responsive (ปรับอัตโนมัติบนมือถือ)

============================================================
